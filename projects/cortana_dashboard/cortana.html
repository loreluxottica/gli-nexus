How to read this dashboard
  • KPI cards — headline totals across all Genie Spaces for the loaded period: messages, active users, queries executed and number of spaces.
  • Message trend over time — one line per space plus a dashed Total line. Hover a point to see exact daily values for every space; click a label in the chart legend to hide/show that space.
  • Usage by space — horizontal bars ranked by message volume; the bar length is relative to the busiest space.
  • User leaderboard — users ranked by total messages. The segmented bar shows how each user splits their activity across spaces (colors match the legend); chips list the per-space message counts.
  • Hover anything (KPI, bar, segment, row) for exact figures.
  • Note: users reach Genie through a shared service principal, so per-user figures are indicative, not exact attribution.
